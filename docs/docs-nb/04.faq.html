❓ Ofte Stilte Spørsmål (OSS)

📄 Hva er Index Page Generator?

Index Page Generator er et PHP-skript designet for å opprette index.html-filer for kataloger. Skriptet gir et strukturert og stilisert gridoppsett som viser innholdet i katalogen. Det inkluderer også inline SVG-ikoner for ulike filtyper og støtter flere språk.

💻 Hvordan kjører jeg Index Page Generator?

For å kjøre skriptet, bruk følgende kommando i terminalen din: 

php ipg.php

Du kan også spesifisere et språk ved å legge til parameteren --lang=xx (f.eks. --lang=nb for norsk).

🔧 Kan jeg tilpasse utdataene fra Index Page Generator?

Ja, du kan tilpasse utdataene ved å redigere konfigurasjonsfilen eller ved å sende parametere gjennom kommandolinjen. Du kan endre layout, ekskludere bestemte filer eller kataloger, og til og med bruke egendefinerte maler.

🌐 Hvilke språk støtter Index Page Generator?

Skriptet støtter følgende språk:

  • Engelsk (en) - Standard
  • Norsk (nb)
  • Filippinsk (fil)
  • Spansk (es)
  • Fransk (fr)
